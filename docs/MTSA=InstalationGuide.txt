Intalation notes (Eclipse Environment)

1. Import MTSA eclipse projects(*) from our MTSA sourceforge project. (https://sourceforge.net/projects/mtsa/)
2. Compile the projects by running the ant task ltsa\build.xml (the default target will compile all the projects) 
3. To run or debug the application use the provided launch: HPWindow.launch

MTSA eclipse projects (*): 
	ltsa 
	MTSTools
	MTSSynthesis
